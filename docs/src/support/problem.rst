常见问题
==================

如果遇到相机使用问题, 请先查阅以下文档

1.相机不出图，驱动报错
------------------
排查方案：
a.检查是不是使用的我们赠送的USB3.0线缆
b.USB连接的是不是电脑的USB3.0口
c.有没有使用USB-hub集线器，本相机不支持使用USB-hub
d.如果以上都不能解决，请使用我们提供的win exe程序，检查相机在win上能否正常出图，如果不能正常出图，请联系客服反厂维修，如果能正常出图，表示相机硬件正常，需要继续排查软件驱动问题

2.相机的vio精度如何
------------------
户外场景实测，300m空旷场景，跑无闭环orbslam3，回到起点误差在几十cm左右，0.1%～0.5%的误差

3.相机是否输出深度图
------------------
c100相机不支持只出深度图，可以通过调用SDK在本地计算输出深度图，此功能正在开发

4.相机是否是彩色相机
------------------
c100相机是灰度相机，c200是彩色相机，正在开发

5.相机支持平台
------------------
Windows、Ubuntu、ROS、ROS2、rk3588、树莓派、orin、xavier

6.相机imu型号
------------------
BMI088
